P4 Proposal: Dice Roll: The Game (now with SCOREBOARD!)

For project 4 I would like to expand on the dice rolling application that I made for project 3. Players will be able to interact with the app as usual but with the additional option of being able to submit their score to a public scoreboard for ranking. This would depend on a MySQL database to keep record of all names and scores, javascript for the application's basic functions, and PHP used within the class framework to process server side tasks related to keeping and displaying scores. Improvements to the dice roll application itself could also be made depending on time constraints as well as the feedback from P3.
