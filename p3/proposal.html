Dice Roll

Rolling dice is a useful way to generate random integers to decide various outcomes related to probability. While you could simply use a random number generator to calculate the same outcome of a dice roll, this does not properly simulate the effect of rolling actual dice. This javascript application will be able to simulate rolling one or several dice with potentially varying side count. Its focus will be on providing a more pleasant asthetic feel than a simple random number generator while still providing the same calculated results.
